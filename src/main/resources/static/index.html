LIST OF APIs:

TYPE API INFO PARAMS EXAMPLE
GET /api/socks Returns the total number of socks in stock that match the query criteria passed in the parameters color - color of socks
operation - comparison operator of the amount of cotton in the composition of socks, one value from: moreThan, lessThan, equal
cottonPart - the value of the percentage of cotton in the composition of socks from the comparison		 /api/socks?color=red&operation=moreThan&cottonPart=90 — must return the total number of red socks with more than 90% cotton
/api/socks?color=black&operation=lessThan?cottonPart=10 — must return the total number of black socks with less than 10% cotton
POST /api/socks/income Add income color - the color of the socks, the string (e.g., black, red, yellow)
cottonPart - the percentage of cotton in the composition of the socks, an integer between 0 and 100 (e.g. 30, 18, 42)
quantity - the number of pairs of socks, an integer greater than 0
POST /api/socks/outcome Add outcome the same as in income

СПИСОК APIs:

ТИП API ИНФО ПАРАМЕТРЫ ПРИМЕРЫ
GET /api/socks Возвращает общее количество носков на складе, соответствующих переданным в параметрах критериям запроса color - цвет носков
operation - оператор сравнения значения количества хлопка в составе носков, одно значение из: moreThan, lessThan, equal
cottonPart - значение процента хлопка в составе носков из сравнения		 /api/socks?color=red&operation=moreThan&cottonPart=90 — должен вернуть общее количество красных носков с долей хлопка более 90%
/api/socks?color=black&operation=lessThan?cottonPart=10 — должен вернуть общее количество черных носков с долей хлопка менее 10%
POST /api/socks/income Добавляет приход color - цвет носков, строка (например, black, red, yellow)
cottonPart - процентное содержание хлопка в составе носков, целое число от 0 до 100 (например, 30, 18, 42)
quantity - процентное содержание хлопка в составе носков, целое число от 0 до 100 (например, 30, 18, 42)
POST /api/socks/outcome Добавляет расход такие же, как и у income запроса
